# 📖 [Rails-React-two-apps] <a name="about-project"></a>

### Preriquisites

NodeJs

### Setup

Clone this repository to your desired folder:
sh
cd my-folder
git clone https://github.com/Roland-Ntwali/Hello-react-front-end

### Install:

Run: npm install
